# Cybersecurity File Encryption Tool 🔐

## Project Overview
This project is a File Encryption Tool built using Python for a cybersecurity internship project.

The tool allows users to encrypt and decrypt files using password-based AES encryption.

## Features
- Secure AES encryption
- Password protected files
- File encryption and decryption
- Works on Android using Pydroid 3

## Technologies Used
- Python
- AES Encryption
- PyCryptodome Library

## How the Tool Works
1. User selects encrypt or decrypt option
2. User enters file path
3. Password is converted to encryption key
4. File is encrypted and saved as .enc file

Example:
photo.jpg → photo.jpg.enc
